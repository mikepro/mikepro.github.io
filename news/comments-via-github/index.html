Comments via github

January 25, 2022

It is now possible for you to add comments to this site via github issues and utterances.

Once you have authorised utterances to comment on your behalf you will be able to comment on any piece of content on this site.

Feel free to say hello below.

Why all the fuss?

This whole website has been generated using hugo. A static site generator.

Hugo takes a series of articles and combines these with a set of HTML templates and produces HTML. No server side languages or databases to configure just simple HTML files which can be hosted pretty much anywhere.

This can prove to be somewhat troublesome for dynamic content like comments. Typically this is normally achieved by having a database and a server side language to process, store and fetch the comments. This is where utterances comes into play. Utterances will take your comment and add it to a github issue for the page. In effect the github issue system becomes a database of comments.

This is quite a neat solution in my humble opinion as it integrates into the existing github estate along with the fact that this website is also hosted using github pages it keeps everything together nicely.
